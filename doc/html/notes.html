safe_float

Notes

safe_float is a headers-only library with few requirements:

  • Library requires C++11.

  • Building of examples and test can be done running b2.

  • Current implementation supports only double as internal Floating Point implementation.

  • Examples have some extra dependencies, including Boost.ProgramOptions.

  • Tests are implemented using Boost.Test and require the compiled version to run, also they depend in Boost.Any and Boost.Rational.
